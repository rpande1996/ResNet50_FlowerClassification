## Overview

This code classifies flower types with the help of a ResNet-50 network.

## Softwares

* Recommended IDE: PyCharm 2021.2

## Libraries

* Numpy 1.21.2
* OpenCV 4.6.0
* SkLearn 1.1.3
* PyTorch 1.12.1+cu116

## Programming Languages

* Python 3.8.12
